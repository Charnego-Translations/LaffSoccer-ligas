# Equipos de fútbol – Charnego Internatiolaff Soccer

Este repositorio contiene una colección de **equipos de fútbol** creados para ser utilizados en el juego **Charnego Internatiolaff Soccer**. También puede usarse con **YSoccer**.

## Estado del proyecto

⚠️ **Proyecto en fase final / pendiente de terminar**

Los equipos incluidos están **prácticamente finalizados**, pero aún pueden contener ajustes pendientes o cambios futuros.
